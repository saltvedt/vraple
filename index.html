Vraple

 
 

What is vraple?

Vraple is a wrapper around search engines. It takes your search query and figures out which search engine is the most well suited to handle it.

Why should I use it?

Because most of the time, you're not searching for information, you're trying to retrieve information you know is available. Search engines should be so good by now that you dont have to look at search results.

What does it actually do?

If the search query ends with an exclamation mark, you are forwarded to the first result on Google. If your search query is mathematical in nature it is sent to WolframAlpha. If it contains !bangs, it is sent to DuckDuckGo. Otherwise, it just sends you to regular Google. If Vraple gets it wrong (like sending you to WolframAlpha when you just wanted to Google something), hitting the backbutton will redirect you to Google (if you're using a supported browser).

Do you have access to my search history?

No, all handling of search queries are done in the client, and cannot be accessed from the server. However, vraple do log everytime you visit the site.


How do I start using Vraple?

Add a new search engine in your browser, using the following URL: http://vraple.com/#q=%s

Examples

!a sicp

Sends you to DDG, which sends you to the Amazon search for "sicp".

\structure and interpretation

Sends you to the first result on DDG.

 

house on the prairie!

Sends you to the first result on Google (using I'm feeling lucky).

123*456

Sends you to Wolfram Alpha.
